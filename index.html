Informations

Configuration requise Systeme

Le parcours cible est zero-prerequis: tu telecharges le launcher, tu doubles-cliques, puis Dreamer Studio prepare la workstation.

  • Windows 10/11 est la cible principale.
  • Node, Python, Rust, pnpm et Tauri ne doivent pas etre requis avant le lancement du setup.
  • Les dependances optionnelles comme SteamCMD, SRCDS, Blender ou Source tools sont detectees puis proposees.
Nouveautes recovery Changelog

Securite & confidentialite Donnees

Cette page ne contient aucun secret et ne doit jamais heberger de backup sensible.

  • Pas de settings.local.json.
  • Pas de fichier .env.
  • Pas de token provider, Steam token, RCON password ou certificat prive.
  • Les mots de passe restent dans Bitwarden ou dans ton coffre local.
Besoin d'aide a l'installation ? Support

Si l'interface finale du launcher n'est pas encore disponible, utilise le runbook d'urgence puis la verification d'integrite.

powershell -NoProfile -ExecutionPolicy Bypass -File tools/installer/verify_release_integrity.ps1 -ReleaseDir <release-folder>

Ce fallback est technique. Le parcours final doit rester un setup graphique autonome.
